# Parcial Final A – Migración a React

Proyecto desarrollado como parte del Parcial Final A de la materia.
Consiste en la migración de un sitio web estático a una aplicación en React utilizando Vite.

## Tecnologías utilizadas
- React
- Vite
- JavaScript
- CSS
- Bootstrap (uso limitado)

## Descripción
El proyecto fue migrado a componentes React, utilizando props, PropTypes y manejo de estado con useState.
Incluye navegación tipo One Page, formulario controlado y estilos separados por componente.

## Ejecución del proyecto
1. Clonar el repositorio
2. Instalar dependencias:
   npm install
3. Ejecutar el proyecto:
   npm run dev
